Bandgap Reference (BGR)

A temperature-independent voltage reference using two BJTs and three resistors with an op-amp.

LTspice Simulation

PCB Design

Description

This circuit combines a PTAT and CTAT voltage to create a stable output reference. The op-amp maintains a constant current across the resistors and BJTs, and the result is a ~1.2V reference.

Key Features

  • PTAT and CTAT cancellation
  • Simulation of temperature sweep in LTspice
  • Designed PCB using KiCad
